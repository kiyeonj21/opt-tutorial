# Optimization Tutorial

## Installation
Install all dependencies listed in [requirements.txt](requirements.txt)

## Run
Run the following opt codes.
``` python
cd ./implementations
python regress_l1.py
python regress_l2.py
python clf.py
python pwl_const.py
python pwl_decre.py
python pwl_speedup.py
python least_l1.py
python dual_qcqp.py
python mc.py
cd ../
```
